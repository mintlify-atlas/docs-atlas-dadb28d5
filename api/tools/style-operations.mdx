title: Style Operations
description: Tools for setting fills, strokes, effects, and visual properties

## set_fill

Set the fill color of a node. Accepts hex (#ff0000) or named color.

<ParamField path="id" type="string" required>
  Node ID
</ParamField>

<ParamField path="color" type="color" required>
  Color value (hex like #ff0000)
</ParamField>

<ParamField path="Returns" type="object">
  Returns the node ID and applied color.
</ParamField>

---

## set_stroke

Set the stroke (border) of a node.

<ParamField path="id" type="string" required>
  Node ID
</ParamField>

<ParamField path="color" type="color" required>
  Stroke color (hex)
</ParamField>

<ParamField path="weight" type="number" default="1">
  Stroke weight (minimum: 0.1)
</ParamField>

<ParamField path="align" type="string" default="INSIDE">
  Stroke alignment. Options: `INSIDE`, `CENTER`, `OUTSIDE`
</ParamField>

<ParamField path="Returns" type="object">
  Returns the node ID, applied color, and weight.
</ParamField>

## set_effects

Set effects on a node (drop shadow, inner shadow, blur). Pass an array or a single effect.

<ParamField path="id" type="string" required>
  Node ID
</ParamField>

<ParamField path="type" type="string" required>
  Effect type. Options: `DROP_SHADOW`, `INNER_SHADOW`, `FOREGROUND_BLUR`, `BACKGROUND_BLUR`
</ParamField>

<ParamField path="color" type="color">
  Shadow color (hex). Ignored for blur.
</ParamField>

<ParamField path="offset_x" type="number" default="0">
  Shadow X offset
</ParamField>

<ParamField path="offset_y" type="number" default="4">
  Shadow Y offset
</ParamField>

<ParamField path="radius" type="number" default="4">
  Blur radius (minimum: 0)
</ParamField>

<ParamField path="spread" type="number" default="0">
  Shadow spread
</ParamField>

<ParamField path="Returns" type="object">
  Returns the node ID and total number of effects.
</ParamField>
